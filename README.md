# Progetto di Visualizzazione delle Informazioni
## Andrea Macale (matricola 560793)

L'applicazione è un sistema di visualizzazione delle informazioni implementato su interfaccia Web, che rappresenta l'occupazione media mensile dei letti, inserendo un intervallo di anni. Come sistema di visualizzazione, è stato implementato un'istogramma a pila, per poter confrontare l'occupazione letti della stessa diagnosi, mese e reparto in due o più anni differenti.
